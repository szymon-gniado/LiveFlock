## Installation

1. Clone the repository and compose:
   
   ```shell
   git clone --recursive git@github.com:szymon-gniado/live-flock.git
   cd live-flock
   docker-compose up
   
5. Access the application at:
   
   ```shell
   http://localhost:3000
   ```
